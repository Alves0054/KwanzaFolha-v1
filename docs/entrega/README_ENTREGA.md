## Ordem recomendada

1. Validar checksum do instalador.
2. Instalar em maquina limpa.
3. Configurar empresa e administrador.
4. Confirmar perfil fiscal com contabilista.
5. Processar folha de teste.
6. Gerar recibo, mapa IRT/INSS e backup.
7. Assinar checklist de entrega.
