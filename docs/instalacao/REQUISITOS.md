# Requisitos

## Cliente

- Windows 10 ou Windows 11, 64 bits.
- 4 GB de RAM minimo; 8 GB recomendado.
- 500 MB livres para instalacao e espaco adicional para base de dados, backups e documentos.
- Conta Windows com permissao para instalar aplicacoes.
- Acesso a internet apenas para ativacao/renovacao quando aplicavel; o funcionamento operacional e local/offline.

## Operacao

- Definir pasta segura para backups.
- Proteger o computador com palavra-passe.
- Manter copias externas dos backups.
- Validar fiscalidade com contabilista antes do primeiro processamento oficial.
