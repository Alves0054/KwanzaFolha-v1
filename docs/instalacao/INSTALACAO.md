# Instalacao

1. Abrir o instalador `KwanzaFolha-Setup-<versao>.exe`.
2. Confirmar o aviso do Windows, caso a build ainda não esteja assinada digitalmente.
3. Escolher a pasta de instalação.
4. Concluir o assistente.
5. Abrir o Kwanza Folha pelo menu iniciar ou atalho no ambiente de trabalho.

## Primeira configuração

1. Criar o utilizador administrador.
2. Configurar os dados da empresa.
3. Confirmar o perfil fiscal em vigor.
4. Cadastrar funcionários.
5. Configurar backups.

## Atualizacao

Antes de atualizar:

- Fazer backup manual.
- Fechar a aplicação.
- Guardar o instalador anterior ate concluir a validação da nova versao.
